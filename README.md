# 프로세스 종료 타이머

Rust와 Ratatui를 사용한 Windows 프로세스 종료 타이머 애플리케이션입니다.

## 기능

- 실행 중인 프로세스 목록 표시
- 프로세스 검색 기능
- 프로세스 선택
- 타이머 설정 (분:초 형식 또는 초 단위)
- 타이머 실행 및 카운트다운 표시
- 지정된 시간 후 프로세스 자동 종료

## 요구사항

- Rust (최신 안정 버전)
- Cargo
- Windows OS

## 빌드 및 실행

```bash
# 빌드
cargo build

# 실행
cargo run

# 릴리즈 빌드
cargo build --release
```

## 사용법

### 프로세스 선택 화면
- `↑` / `↓`: 프로세스 목록에서 이동
- `/`: 검색 모드 활성화 (검색어 입력)
- `Enter`: 선택한 프로세스 선택
- `Q`: 앱 종료

### 타이머 설정 화면
- 타이머 시간 입력:
  - 분:초 형식: `5:30` (5분 30초)
  - 초 단위만: `300` (300초)
- `Enter`: 타이머 시작
- `Esc`: 취소하고 프로세스 선택 화면으로 돌아가기
- `Q`: 앱 종료

### 타이머 실행 화면
- 남은 시간이 실시간으로 표시됩니다
- `Q`: 타이머 취소

## 주의사항

- 프로세스 종료는 시스템에 영향을 미칠 수 있으므로 주의하세요
- 관리자 권한이 필요한 프로세스의 경우, 관리자 권한으로 실행해야 합니다
- 중요한 시스템 프로세스를 종료하지 않도록 주의하세요

## 의존성

- `ratatui`: 터미널 UI 라이브러리
- `crossterm`: 크로스 플랫폼 터미널 조작 라이브러리
- `sysinfo`: 시스템 정보 및 프로세스 관리 라이브러리
